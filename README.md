# Frontend-Experiment8: Countdown Timer (JavaScript setInterval)

## Objective
Build a simple countdown timer that displays the remaining time in days, hours, minutes, and seconds, updating every second until the target date is reached.

## Files
- `index.html` - Main HTML page containing the timer layout.
- `style.css` - Styles for the timer and page.
- `script.js` - JavaScript code handling the countdown logic.

## How to Run
1. Clone or download this repository.
2. Open `index.html` in any modern web browser.
3. The countdown timer will automatically start and display the remaining time until **September 28, 2025, 23:59:59**.
4. When the countdown reaches zero, a message "Timeâ€™s up!" will be displayed.

## Features
- Displays days, hours, minutes, and seconds.
- Updates every second using `setInterval()`.
- Handles the countdown completion.
